A while loop runs its body over and over as long as the condition is True. The moment count reaches the threshold, the condition becomes False — and the loop stops. Try increasing the threshold, then watch what happens if you forget to increment count: the condition never becomes false and the loop runs forever.

count = 0
while count < 5:
    print(count)
    count = count + 1

A list comprehension builds a new list by looping through an existing one and (optionally) filtering items. Read it left to right: x is each item, for x in numbers is the loop, if x % 2 == 0 is the filter — only even items make it through. Items that fail the condition are dropped (shown in red). Items that pass land in the result list (shown in teal).

Try changing the list and filter above to see how the same pattern applies to different data.
