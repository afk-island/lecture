What it is

Most platforms separate learning from teaching. Lecture doesn't. The act of learning well produces something worth sharing. Here, you move between student and teacher naturally — sometimes in the same session.

Tracks, not courses
A track is a structured sequence of lessons built around a framework — yours. Link a YouTube playlist, describe a method, or port a curriculum. AI handles the exercise generation and structure. You handle the insight.
Forums that fossilize
Discussions are private to people who've completed a given track. They close after 50 posts and become permanently citable.
Any subject
The first track is 7×7 Code — seven programming languages, seven weeks, one concept at a time. But the format works for mathematics, physics, history, music theory, or anything else with structure worth sharing.
Your API key
Lecture uses the Anthropic API for exercise grading and generation. You bring your own key. Your data stays in your browser. No accounts, no subscriptions, no intermediary.
Try the prototype
The demo is a working version of a 7×7 Code track — Week 1, covering variables, conditionals, and loops across Python, TypeScript, SQL, Rust, Go, R, and Zig. You can complete lessons, get AI grading on your exercises, and see how the creator workflow feels.
This is an prototype built by one person to explore an idea. Accounts, the map view, forums, and multi-track browsing are not yet functional. Exercise generation requires an Anthropic API key. Some things will break. That's the point of showing it early.
